## 🎓 Học Từ Prompts

### Cách tối ưu prompts:
1. **Be Specific** - Mô tả rõ ràng
2. **Include Context** - Cho biết ngữ cảnh
3. **List Requirements** - Liệt kê chi tiết
4. **Ask Why** - Hiểu lý do đằng sau

### Ví dụ:
```
❌ "Tạo form"

✅ "Create a blog post form with:
   - Fields: title, content, excerpt
   - Validation: title required, min 100 chars content
   - Image upload support
   - Status selector (draft/published)
   - Error messages display
   - Success redirect to dashboard"
```
